# Candy Client Rewrite (Crackloader)

---

## 🇯🇵 日本語

### 📥 使い方
- **リリース** または **ソースコード** の `patched` フォルダにある `crackloader.jar` を Minecraft の `mods` フォルダにコピーしてください。  
- Click GUI のキー: **Y**

### 📥 Usage
